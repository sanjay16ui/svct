# Larkspur Crochets

## Setup (first time only)
```bash
git clone <repo-url>
cd svct-main
npm install
cd backend && npm install && node reset-admin.js && cd ..
```

## Run the project
```bash
# Terminal 1 — backend
cd backend
node server.js

# Terminal 2 — frontend
npm run dev
```

## Check the database (VS Code terminal)
```bash
cd backend
node dbcheck.cjs
```
